Buttons

Bootstraps contains several different types of buttons. "btn btn-XXXXX" where XXXXX is the type of button.
Types:
  • primary
  • secondary
  • success
  • danger
  • warning
  • info
  • light
  • dark
  • link


<button type="button" class="btn btn-primary">Primary</button> <button type="button" class="btn btn-secondary">Secondary</button> <button type="button" class="btn btn-success">Success</button> <button type="button" class="btn btn-danger">Danger</button> <button type="button" class="btn btn-warning">Warning</button> <button type="button" class="btn btn-info">Info</button> <button type="button" class="btn btn-light">Light</button> <button type="button" class="btn btn-dark">Dark</button> <button type="button" class="btn btn-link">Link</button>
Buttons can also be used with other tags like <a> and <input>.

Link <a class="btn btn-primary" href="#" role="button">Link</a> <button class="btn btn-primary" type="submit">Button</button> <input class="btn btn-primary" type="button" value="Input">
"btn btn-outline-XXXXX" does the same thing as "btn btn-XXXXX" but with an OUTLINE.

<button type="button" class="btn btn-outline-primary">Primary</button>
